Sobre el proyecto

"The Real Code Wars" es un proyecto de fin de Grado de la USAL en colaboración con HP.
Se trata de una plataforma en la que los usuarios pueden subir código en python utilizando la biblioteca proporcionada (RealCodeWars.py).
Ese código controlará los soldados en una guerra.
Se celebrarán batallas entre distintos equipos y ganará el que primero destruya la base enemiga.
